kali 安装w3af踩坑记录

首页 信息安全 Kali

w3af 是 web application attack and audit framework (web 应 用攻击和安全审计框架) 的缩写。

他是一个开源的 Web 应用安全扫描器和漏洞利用工具

w3af 文档:http://docs.w3af.org/en/latest/

w3af 仅支持 Python2.x 版本，不支持 3.x

# 下载 w3af

┌──(jiangjiyue ㉿ kali)-[/opt/ 漏洞分析]
 └─$ cd /opt/ 漏洞利用 /
 ┌──(jiangjiyue ㉿ kali)-[/opt/ 漏洞分析]
 └─$ sudo git clone --depth 1 https://github.com/andresriancho/w3af.git

# 查看 Python 版本

# w3af 只支持 2.x 版本的 python
 ┌──(jiangjiyue ㉿ kali)-[/opt/ 漏洞分析]
 └─$ python --version
 # 如果是 3.x 的版本，再用下面这条命令看 kali 上面装了几个 python 版本，一般是 3 个
 ll /usr/bin grep python
 # 切换 python 版本
 ┌──(jiangjiyue ㉿ kali)-[/opt/ 漏洞分析]
 └─$ update-alternatives --install /usr/bin/python python /usr/bin/python2.7 1
 update-alternatives: 使用 /usr/bin/python2.7 来在自动模式中提供 /usr/bin/python (python)
 ┌──(jiangjiyue ㉿ kali)-[/opt/ 漏洞分析]
 └─$ python --version
 Python 2.7.18

# 安装 PIP

┌──(jiangjiyue ㉿ kali)-[/opt/ 漏洞分析 /w3af]
 └─$ sudo wget https://bootstrap.pypa.io/pip/2.7/get-pip.py        
 ​
 ┌──(jiangjiyue ㉿ kali)-[/opt/ 漏洞分析 /w3af]
 └─$ sudo python get-pip.py
 ​
 ┌──(jiangjiyue ㉿ kali)-[/opt/ 漏洞分析 /w3af]
 └─$ sudo pip config set global.index-url https://pypi.tuna.tsinghua.edu.cn/simple        
 ​
 ┌──(jiangjiyue ㉿ kali)-[/opt/ 漏洞分析 /w3af]
 └─$ sudo pip install jsonschema==2.6.0
 ​
 ┌──(jiangjiyue ㉿ kali)-[/opt/ 漏洞分析 /w3af]
 └─$ sudo bash /tmp/w3af_dependency_install.sh

# 安装 Npm

┌──(jiangjiyue ㉿ kali)-[/opt/ 漏洞分析 /w3af]
 └─$ sudo apt-get install npm
 ​
 ┌──(jiangjiyue ㉿ kali)-[/opt/ 漏洞分析 /w3af]
 └─$ sudo npm install -g retire@2.0.3
 ​
 ┌──(jiangjiyue ㉿ kali)-[/opt/ 漏洞分析 /w3af]
 └─$ sudo npm update -g retire

# 启动 w3af

┌──(root ㉿ kali)-[/opt/ 漏洞分析 /w3af]
 └─# chown root:root /tmp/w3af_dependency_install.sh
 ​
 ┌──(root ㉿ kali)-[/opt/ 漏洞分析 /w3af]
 └─# vim w3af_console
  # 注释掉 dependency_check () 函数，在第 13 行 不注释有的即使装了环境也无法启动

┌──(root ㉿ kali)-[/opt/ 漏洞分析 /w3af]
 └─# ./w3af_console

 w3af>>> exit

# 设置环境变量

┌──(jiangjiyue ㉿ kali)-[/opt/ 漏洞分析 /w3af]
 └─$ sudo vim /etc/profile
 # 添加在最后一行
 export PATH=PATH:/opt/漏洞分析/w3af  ┌──(jiangjiyue㉿kali)-\[/opt/漏洞分析/w3af\]  └─ sudo source /etc/profile
