geoip

geoip 是 object 类型，它有几个子字段，含义如下：

  • geoip.city_name：城市
  • geoip.continent_name：大陆名称
  • geoip.country_iso_code：国家编码
  • geoip.location：经纬度坐标，必须是：geo_point 类型
  • geoip.region_iso_code：地域编码
  • geoip.region_name：地域名称
